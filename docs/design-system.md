# Design System

## Visual direction
- Minimal, typography-led reading experience.
- Warm neutral light theme and graphite dark theme.
- Subtle elevation and restrained borders.

## Stack
- Tailwind CSS for tokenized styling.
- Base UI primitives for accessible behavior.
- Motion for intentional transitions.

## Typography
- Display: Newsreader.
- UI/body: Manrope.
- Mono: JetBrains Mono.

## Motion
- Page enter transitions, sidebar expansion, command dialog transitions, TOC highlight.
- Global reduced-motion support through a single motion preference gate.

## Accessibility baseline
- Keyboard navigation for all interactive navigation/search components.
- Visible focus rings.
- WCAG-compliant contrast targets.
